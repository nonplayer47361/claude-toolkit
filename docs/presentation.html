Rate Limit 병목

상황에 맞는 패러다임 선택

외부 CLI가 있으면 패러다임 A, Claude Code만 있으면 패러다임 B

패러다임 A

외부 CLI 에이전트

Codex(GPT-4o)·agy(Gemini)를 Claude의 서브 에이전트로 배정. 파일 기반 통신으로 병렬 실행 가능.

🧠 Claude (오케스트레이터) 🤖 Codex ✨ agy
  • Rate limit 분산 — 에이전트별 독립 리밋
  • codex + agy 병렬 배정으로 속도 향상
  • verify.sh 5-point 자동 품질 게이트
  • AgentShield 7카테고리 보안 스캔
  • 적응형 점수 기반 에이전트 자동 선택
패러다임 B

내부 서브에이전트

Claude Code만으로 planner·qa-analyst·impl-dev 역할의 서브에이전트를 내부에서 생성.

🧠 Claude (오케스트레이터) 🔧 내부 서브에이전트
  • 별도 CLI 설치 없이 바로 시작 가능
  • Agent() 도구로 역할 분리·병렬 호출
  • _workspace/ 파일 + SendMessage 통신
  • QA 에이전트가 직접 결과 검증
  • harness 스킬과 연동
스킬 3종

설치하면 즉시 쓸 수 있는 능력

원라이너 설치 후 Claude Code에서 바로 호출 가능

📝
git-helper

커밋 메시지 & PR 초안

git diff를 읽어 커밋 메시지를 자동 생성하고, PR 초안까지 작성해준다. "커밋 메시지 써줘" 한 마디면 충분.

.\scripts\install-skill.ps1 -SkillName git-helper
🔍
code-review-ko

한국어 코드 리뷰

diff를 분석해 보안·가독성·성능 관점의 지적사항을 한국어로 정리한다. "코드 리뷰해줘"로 실행.

.\scripts\install-skill.ps1 -SkillName code-review-ko
🚀
cli-agent-team

다중 에이전트 루프

Claude 오케스트레이터 + Codex/agy 서브에이전트. 아이디어 회의부터 배포까지 자동화 루프를 구성한다.

.\scripts\install-skill.ps1 -SkillName cli-agent-team
핵심 기능

자동화를 신뢰할 수 있게 만드는 장치들

🛡️

AgentShield

7카테고리 보안 스캐너. API 키 노출·위험 명령어·코드 인젝션 패턴을 커밋 전 자동 검사.

✅

5-Point Verify Gate

스코프 초과·AC 미완료·보안 이슈·증거 파일 누락·자동 커맨드 실패를 한 번에 검증.

🔗

ECC 세션 연속성

세션이 바뀌어도 .session_state를 자동 주입. 어제 하던 작업을 오늘 "계속"으로 재개.

📊

적응형 에이전트 배분

task_type별 AC 통과율을 학습해 다음 배정에 반영. 데이터가 쌓일수록 정확해진다.

⚡

병렬 배정

독립 태스크를 codex+agy에 동시 배정. parallel-check 안전성 검사 통과 시에만 실행.

🌉

pty-bridge

Windows 비-TTY 환경에서 agy가 출력을 내지 않는 문제를 Node.js 브릿지로 해결.

5분 안에 시작

지금 바로 설치하기

원라이너 설치 후 Claude Code를 재시작하면 스킬이 활성화된다

설치 (Windows)

원라이너로 모든 스킬을 한번에 설치

irm https://raw.githubusercontent.com/nonplayer47361/claude-toolkit/main/scripts/install.ps1 | iex

설치 (macOS/Linux)

curl로 동일하게 설치 가능

curl -fsSL https://raw.githubusercontent.com/nonplayer47361/claude-toolkit/main/scripts/install.sh | bash

에이전트 설정

프로젝트에서 에이전트 감지 및 설정 파일 생성

bash ~/.claude/skills/cli-agent-team/scripts/agent-team.sh doctor

첫 태스크 실행

TASK.md 작성 후 에이전트 배정

agent-team.sh dispatch codex T001 limited "$(pwd)" execute
